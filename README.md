# Breaking Bad Character Search Application

It is an app that you can search your favorite characters from Breaking Bad tv show. You can your favorite characters to your list and delete them. The app utilizes Context API and Reducer with useReducer Hook. It also uses LocalStorage  React Router DOM. In addition it uses Framer Motion animations. I fetch the data from free Breaking Bad API.

Here are the full list of tecnologies I utilized:

* ReactJS

* SASS

* Javascript

* Material-Ui

* Framer Motion

* React-Router-Dom

* Axios
